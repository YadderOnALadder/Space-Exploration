Lucy: NASA Mission

Lucy was the first ever NASA space mission to locate Trojan asteroids. These types of asteroids contain clues to cracking the code on how the solar system was created. This mission was launched on October 16, 2021.

Lunar Reconnaissance Orbiter (LRO)

The Lunar Reconnaissance Orbiter (LRO) mission was for mapping out the moon's lunar surface like nobody has done before. This mission was launched on June 18, 2009.
